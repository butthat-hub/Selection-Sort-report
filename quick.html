簡述原理

快速排序是最知名的高效演算法，同樣採用分而治之的策略。

核心動作是分割 (Partition)：在陣列中挑選一個基準值 (Pivot)，然後重新排列陣列，讓所有比 Pivot 小的元素移到它的左邊，比它大的移到右邊。這時 Pivot 就已經在最終正確的位置上了。接著對左右兩半重複此動作。

演算法邏輯

void quickSort(arr, low, high) {
  if (low < high) {
    // 取得基準點就位後的索引
    pi = partition(arr, low, high)
    
    // 對左右兩側進行遞迴
    quickSort(arr, low, pi - 1)
    quickSort(arr, pi + 1, high)
  }
}
