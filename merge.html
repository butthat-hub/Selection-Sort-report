🧬 合併排序法 Merge Sort

模擬實作操作台

準備就緒

簡述原理

合併排序採用分而治之策略。首先將陣列遞迴地切分成兩半，直到每個子陣列只剩一個元素（此時自然已排序）。接著執行「合併」動作：將兩個已排序的子陣列合併成一個更大的、已排序的陣列，直到合併回原始長度。

複雜度分析

情況時間複雜度
最佳 / 最壞O(n log n)
空間複雜度O(n)

核心邏輯 (C++)

void mergeSort(int a[], int l, int r) {
  if (l < r) {
    int m = l + (r - l) / 2;
    mergeSort(a, l, m);
    mergeSort(a, m + 1, r);
    merge(a, l, m, r);
  }
}
