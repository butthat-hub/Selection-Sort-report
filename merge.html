簡述原理

合併排序法採用分而治之 (Divide and Conquer) 策略。

首先將陣列不斷從中間對切，直到每個子陣列只剩下一個元素（此時視為已排序）。接著執行「合併」：比較兩個已排序的子陣列的第一個元素，將較小的拿出放入暫存陣列，不斷重複直到兩個子陣列完美合併成一個較大的有序陣列。

演算法邏輯

void mergeSort(arr, L, R) {
  if (L < R) {
    mid = L + (R - L) / 2
    mergeSort(arr, L, mid)
    mergeSort(arr, mid+1, R)
    merge(arr, L, mid, R)
  }
}
