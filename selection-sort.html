簡述原理

選擇排序法的邏輯非常直觀，就像是在一堆數字中「挑選」出最小的：

將陣列分為「已排序」與「未排序」兩個部分。每一輪都會從「未排序」的範圍中，完整掃描一遍以找出最小值。找到最小值後，將它與未排序範圍的第一個元素交換位置。這樣一來，最小值就被納入了已排序的範圍。重複此動作直到全部排完。

演算法邏輯 (虛擬碼)

for i from 0 to n-1:
  min_idx = i
  // 在未排序區塊尋找最小值
  for j from i+1 to n:
    if arr[j] < arr[min_idx]:
      min_idx = j
      
  // 將最小值與未排序的首個元素交換
  swap(arr[i], arr[min_idx])
