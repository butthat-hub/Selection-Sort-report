氣泡排序法 (Bubble Sort)

1. 演算法原理

從第一個元素開始，比較相鄰的兩個元素，如果前一個比後一個大則交換，重複此動作直到最後。每一輪都會將目前未排序中最大的數移到正確位置。

2. 演示過程 (範例數列: [5, 3, 8, 2])

第一輪：

  • [5, 3, 8, 2] → 比較 5, 3 (5 > 3，交換) → [3, 5, 8, 2]
  • [3, 5, 8, 2] → 比較 5, 8 (不交換)
  • [3, 5, 8, 2] → 比較 8, 2 (8 > 2，交換) → [3, 5, 2, 8]

第二輪：

  • [3, 5, 2, 8] → 比較 3, 5 (不交換)
  • [3, 5, 2, 8] → 比較 5, 2 (5 > 2，交換) → [3, 2, 5, 8]

第三輪：

  • [3, 2, 5, 8] → 比較 3, 2 (3 > 2，交換) → [2, 3, 5, 8] (排序完成)

3. 複雜度分析

時間複雜度：$O(n^2)$，空間複雜度：$O(1)$。
