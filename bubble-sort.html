一、操作原理

氣泡排序法（Bubble Sort）是一種簡單的比較排序演算法。
核心概念：重複走訪要排序的數列，一次比較兩個相鄰的元素，如果它們的順序錯誤就交換過來。

走訪數列的工作是重複地進行直到沒有再需要交換，也就是該數列已經排序完成。 這個演算法的名字由來是因為越小的元素會經由交換慢慢「浮」到數列的頂端（或大元素沉到底部）。

二、演算法偽代碼

for i = 0 to n-2:
  swapped = false
  for j = 0 to n-2-i:
    if arr[j] > arr[j+1]:
      swap(arr[j], arr[j+1])
      swapped = true
  if not swapped: break

三、時間 & 空間複雜度

情況 時間複雜度 說明
最佳情況 O(n) 當輸入資料已經是正序時（最佳化版本）
平均/最壞 O(n²) 需要兩層迴圈處理
空間複雜度 O(1) 原地排序，僅需常數級額外空間
