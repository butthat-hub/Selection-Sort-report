簡述原理

氣泡排序是一種最基礎的比較排序演算法。

運作方式是重複掃描陣列，每次比較相鄰的兩個元素。如果它們的順序錯誤（例如前一個比後一個大），就將它們交換。透過每一輪的掃描，陣列中未排序的最大值就會像水中的氣泡一樣，逐漸「浮」到陣列的最右側。

演算法邏輯 (優化版虛擬碼)

for i from 0 to n-1:
  swapped = false
  for j from 0 to n-i-2:
    if arr[j] > arr[j+1]:
      swap(arr[j], arr[j+1])
      swapped = true
  // 若此輪無交換，代表已排好
  if not swapped: break
