氣泡排序法 (Bubble Sort)

1. 簡述原理

重複走訪要排序的數列，一次比較兩個元素，如果順序錯誤就交換，直到沒有任何交換發生。

2. 複雜度分析

  • 最佳時間複雜度：$O(n)$
  • 平均時間複雜度：$O(n^2)$
  • 空間複雜度：$O(1)$

3. 模擬實作內容

// C 語言範例
for (int i = 0; i < n - 1; i++) {
    for (int j = 0; j < n - i - 1; j++) {
        if (arr[j] > arr[j + 1]) {
            int temp = arr[j];
            arr[j] = arr[j + 1];
            arr[j + 1] = temp;
        }
    }
}
