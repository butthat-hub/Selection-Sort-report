簡述原理

插入排序的邏輯就像是在打撲克牌時整理手牌：

我們會將陣列分為「已排序」與「未排序」兩個部分。每次從未排序的部分中抽出第一個元素（我們稱之為 Key），由右至左與已排序的元素進行比較。如果已排序的元素比 Key 大，就把該元素往右移一格，直到找到適合的位置將 Key 插入。

演算法邏輯 (虛擬碼)

for i from 1 to n-1:
  key = arr[i]
  j = i - 1
  while j >= 0 and arr[j] > key:
    arr[j + 1] = arr[j]  // 右移
    j = j - 1
  arr[j + 1] = key       // 插入
